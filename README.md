 School Live Map Application
יישום לניהול בית ספר הכולל מערכת הרשמת תלמידות, כניסת מורים מאובטחת, ומעקב מיקום חי וחכם על גבי מפה.

 תכונות מרכזיות
ממשק משתמש (UI): עמוד בית המאפשר רישום קל של תלמידות וכניסה למורים.

מערכת איכון מבוססת מפה: שילוב של מפת Leaflet להצגת מיקומים בזמן אמת של התלמידות ושל המורה.

התראות חכמות: זיהוי אוטומטי ואזהרה ויזואלית (סמן אדום) עבור תלמידות שהתרחקו מעבר לרדיוס של 3 ק"מ ממיקום המורה.

מסד נתונים: ניהול יעיל של נתוני התלמידות, המורים והמיקומים באמצעות MongoDB.

צד שרת ותצוגה: שרת Express המשלב תבניות EJS לרינדור דינמי של דפי המערכת.

 דרישות מקדימות
טרם התקנת הפרויקט, יש לוודא שהכלים הבאים מותקנים בסביבת העבודה שלך:

Node.js (כולל מנהל החבילות npm)

מסד נתונים MongoDB (רץ מקומית על המחשב או בענן דרך MongoDB Atlas)

 התקנה והרצה
1. התקנת הפרויקט
פתח את המסוף (Terminal) בתיקיית הפרויקט והרץ את הפקודה הבאה להתקנת כל התלויות:

Bash
npm install
2. הגדרת משתני סביבה (.env)
צור קובץ בשם .env בתיקיית השורש של הפרויקט (ליד הקובץ app.js) והגדר בו את המשתנים הבאים:

קטע קוד
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
(יש להחליף את הערכים בפרטי החיבור והסודות האמיתיים שלך).

3. הקמה והרצת מסד הנתונים
בסביבה מקומית: יש לוודא ששירות ה-MongoDB רץ. בסביבת Windows ניתן להשתמש בפקודה net start MongoDB או להריץ mongod --dbpath <path>.

בענן: יש לוודא שמחרוזת ה-URI ל-MongoDB Atlas מעודכנת בקובץ ה-.env.

4. הפעלת השרת
להרצת השרת בסביבת ייצור:

Bash
npm start
להרצת השרת בזמן פיתוח (באמצעות nodemon, במידה ומותקן):

Bash
npm run dev
5. צפייה במערכת
פתח את הדפדפן וגש לכתובת:
http://localhost:5000

 מבנה תיקיות
app.js - נקודת הכניסה של השרת והגדרות הבסיס.

routers/ - ניהול הנתיבים (Routes) לסטודנטים, מורים ומיקומים.

controllers/ - הלוגיקה העסקית וטיפול בבקשות הלקוח.

models/ - הגדרת מודלים וסכמות עבור השמירה ב-MongoDB.

views/ - תבניות צד-שרת (EJS) המייצרות את דפי התצוגה.

 תלויות חיצוניות וספריות (Dependencies)
express - תשתית שרת ה-HTTP וניהול ה-API.

ejs - מנוע רינדור תבניות צד-שרת.

mongoose - ספריית ה-ORM לעבודה נוחה מול מסד הנתונים MongoDB.

dotenv - טעינה וניהול של משתני סביבה רגישים.

cookie-parser - חילוץ וקריאת עוגיות מהבקשות (Cookies).

jsonwebtoken - יצירה ואימות של טוקנים מבוססי JWT לאבטחה.


 שירותים נדרשים
MongoDB - מסד הנתונים העיקרי לשמירת מידע על מורים, תלמידות והיסטוריית מיקומים.

Node.js - סביבת הריצה עליה יושב שרת ה-Express.
